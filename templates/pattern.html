Creating a Pattern

To create your own template, you will need to enter the template name followed by the words keys separated by semicolons. It is important that the word order of the keys that the user enters matches the order in the document.

Example: Pattern name;Profile number;Last name;First Name

Example(rus): Название шаблона;Номер профиля;Фамилия;Имя
